All Commands

14 commands to handle any development task. Filter by category or search with ⌘K.

/feature Workflow

Implement a new feature with full workflow support

Example: /feature "Add user authentication"
Plan → Implement → Test → Review
/bugfix Workflow

Fix a bug with diagnosis and structured workflow

Example: /bugfix "API returns 500 on checkout"
Debug → Test → Fix → Verify
/plan Workflow

Create implementation plan only (no coding)

Example: /plan "Refactor auth module"
Output: .claude/.plans/[datetime]-plan.md
/implement Workflow

Execute an existing implementation plan

Example: /implement .claude/.plans/plan.md
/discuss Workflow

Structured discussion for requirements and decisions

Example: /discuss "REST vs GraphQL for API?"
Output: Discussion Summary + ADR
/review Quality

Review code changes for quality and security

Example: /review src/auth/
/test Quality

Write tests for specified code

Example: /test src/services/user.ts
/debug Quality

Debug issue and find root cause (diagnosis only)

Example: /debug "Memory leak in dashboard"
/refactor Quality

Refactor code safely without changing behavior

Example: /refactor src/utils/helpers.ts
/check Quality

Run lint, typecheck, and tests

Example: /check --fix
/commit Git

Create a git commit with conventional message

Example: /commit
/pr Git

Create a pull request with proper description

Example: /pr "Add authentication"
/onboard Setup

Analyze project and recommend Claude Code setup

Example: /onboard --apply
Detects: Languages, frameworks, databases, CI/CD

Get Started

Install in 30 Seconds

Copy the .claude directory to your project and you're ready to go. It's that simple.

1

Clone the Repository

git clone https://github.com/anthropics/claude-code-kit.git
2

Copy to Your Project

cp -r claude-code-kit/.claude /path/to/your/project/
3

Run Onboarding

/onboard --apply

This detects your stack and creates optimized rules for your project

Requirements

  • Claude Code (desktop or CLI)
  • Git (for /commit and /pr)
  • GitHub CLI (optional, for /pr)
